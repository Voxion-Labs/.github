## 🧠 Core Focus
Systems Programming (C/C++) • WebAssembly Architectures • AI Operating Systems • Zero-Backend Networks • LLM Security

---

## 🧪 Voxion eXperimental Research (VXR)
*Dedicated to browser-native, zero-backend, and deterministic memory isolation architectures.*

* **[VXR-Sandbox]** — A browser-native, deterministic sandboxing kernel designed to detect and block LLM prompt injections locally.
* **[VXR-Continuum]** — A zero-backend, purely client-side engine demonstrating Distributed State Synchronization across edge networks using CRDT Cookies.
* **[Aegis-IR]** — A browser-native, zero-backend search engine prototype utilizing C++ WebAssembly for deterministic memory isolation.

---

## ⚙️ Core Systems & AI Platforms
* **[VoidLAB]** — A cloud code editor and compiler featuring a modern IDF-style workspace and multi-language execution.
* **[FluxKernel]** — An AI Operating System Agent that acts as a persistent, memory-driven kernel operating directly on the filesystem.
* **[RedTerminal]** — A lightweight systems programming project written in C that simulates process lifecycle management and scheduling.

---

## 🌐 Ecosystem & Applications
* **[HorizonSync]** — A unified workspace designed to eliminate app fatigue by seamlessly combining real-time communication hubs.
* **[LEXIO]** — A Premium Text-To-Speech (TTS) web application powered by the Web Speech API with real-time highlighting.
* **[SkillNODE]** — A modern multi-page skill development platform combining competitive practice and AI-assisted learning.
* **[Empty_Pointer]** — A grid-based action-survival game built natively in C++ where you navigate the system and evade Red-Code algorithms.
